🟢 Conexión a Supabase

Pega las credenciales de tu proyecto Supabase:

¿Cómo obtener tus credenciales?
  1. Entra a supabase.com y crea tu proyecto gratis
  2. Ve a Project Settings → API
  3. Copia la Project URL y la anon public key
  4. En Table Editor, crea una tabla llamada pedidos
  5. Pega todo aquí arriba y haz clic en Guardar y Conectar

Columnas requeridas en la tabla pedidos:

id (int8 PK) · created_at (timestamptz) · fecha (text) · hora (text) · cliente (text) · telefono (text) · pedido (text) · entrega (text) · direccion (text) · pago (text) · total (text)
